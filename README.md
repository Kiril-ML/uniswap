# Uniswap-clone

## Description

Clone of [uniswap](https://app.uniswap.org/) for education purpose.

Functional:

- swap pairs;
- add of the pair's liquidity;
- withdrawal of liquidity;
- registration of a token pair that does not exist yet;

## Demo

[demo](https://uniswap-pearl-phi.vercel.app/)

### Installation

```bash
git clone https://github.com/Kiril-ML/uniswap.git
cd uniswap
npm i
```
